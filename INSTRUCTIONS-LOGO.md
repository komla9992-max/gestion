# Instructions pour ajouter le logo CGSP

## Fichier requis

Pour que le logo s'affiche correctement dans l'application, vous devez ajouter le fichier image du logo dans le dossier :

```
public/images/logoCGSP.png
```

## Formats acceptés

Le code accepte maintenant les formats suivants (dans cet ordre de priorité) :
1. `logoCGSP.png` (recommandé)
2. `logoCGSP.jpg` (si PNG non disponible)

## Dimensions recommandées

Le logo sera automatiquement redimensionné, mais pour un meilleur rendu :
- **Format** : PNG avec fond transparent ou JPG
- **Résolution** : Minimum 200x240 pixels (pour un logo en forme de bouclier)
- **Ratio** : Le logo sera adapté automatiquement au conteneur

## Emplacements où le logo s'affiche

1. **Sidebar (Navigation)** - Dimensions : 90x100px
2. **Page de connexion** - Dimensions : 120x140px  
3. **Exports PDF** (Factures, rapports) - Dimensions : 40x40px par défaut

## Si le logo ne s'affiche pas

1. Vérifiez que le fichier est bien dans `public/images/`
2. Vérifiez que le nom du fichier est exactement `logoCGSP.png` ou `logoCGSP.jpg`
3. Vérifiez que le fichier n'est pas corrompu
4. Ouvrez la console du navigateur (F12) pour voir les erreurs éventuelles

## Test

Après avoir ajouté le fichier, rafraîchissez la page (Ctrl+F5 pour forcer le rechargement).
